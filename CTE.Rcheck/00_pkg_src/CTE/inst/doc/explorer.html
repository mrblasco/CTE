The function returns a list of interventions matched with the name bortezomib.

Each intervention is associated with a unique clinical trial ID, title, status, date of registration, a detailed description of the study, and the number of enrolled subjects (either expected or actual) for 53 clinical trials (out of 54 entries).

I use the function datatable from the package DT to show the results as HTML table.

DT::datatable(dplyr::select(tbl, -enrollment,-enrollment_type)
    , rownames = FALSE
    , filter = "top"
    , extensions = "Buttons"
    , plugins = "ellipsis"
    , options = list (
        dom = 'Bfrtip'
        , buttons =  c('copy', 'csv')
        , columnDefs = list(list(
                targets = c(2, 5)
                , render = JS("$.fn.dataTable.render.ellipsis( 125 )")
                )
            )
        )
    )
